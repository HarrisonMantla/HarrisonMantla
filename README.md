## Projects

### Avalon

Avalon is a local-first AI command system built to reduce friction between intention and action.

It is designed around:

* agents
* routines
* research tools
* memory-gated workflows
* approval-first actions
* personal mission control

**Core principle:** *Minimize friction. Never remove consent.*

Avalon may help, suggest, draft, open, search, and organize.

Avalon must ask before sending, deleting, buying, posting, messaging, or changing anything important.
